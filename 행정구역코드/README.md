# 행정구역 코드표 (SGIS 기준) — 작업기록 & 이어가기

> 작성: 2026-07-08 · 목적: 복합쇠퇴진단 **전국출력**용 시도/시군구/행정동 코드-명칭 조인 마스터표
> 다음에 이어서 작업하려면 이 파일부터 읽으면 됨.

## 1. 무엇을 만들었나
SGIS(통계지리정보) Open API로 **전국 행정구역 코드-명칭 표**를 자동 생성해 엑셀로 저장.
- 구조: `연번 | 시도코드 | 시도명 | 시군구코드 | 시군구명 | 행정동코드 | 행정동명`
- 규모: 전국 **3,559개 행정동** (2026-07-08 기준)

### 산출물
| 파일 | 내용 |
|---|---|
| `행정구역코드_전국.xlsx` | 통합본. `전체` 시트(전국 한 방에) + 시도별 시트 17개 = **총 18시트** |
| `시도별파일/` | 시도 하나당 엑셀 1개씩 = **17개 파일** (예: `35_전북특별자치도.xlsx`). 파일마다 연번 1부터 새로 매김 |
| `sgis_admin_codes.py` | 위 엑셀들을 만드는 생성 스크립트 (재실행하면 최신본으로 갱신) |

## 2. ⚠️ 코드 체계 주의 (제일 중요)
여기 코드는 **통계청(SGIS) 체계**다. 행안부 행정표준코드와 **다르다**.

| 체계 | 전북 | 성격 |
|---|---|---|
| **통계청/SGIS (이 표)** | **35** | 인구총조사·집계구·SGIS 경계·쇠퇴진단에서 씀 |
| 행안부 행정표준코드 | 45 | 법정동/행정동 표준(주소·행정). code.go.kr |

- 코드는 **계층 포함** 구조라 조인이 쉬움:
  시도 `35` → 시군구 `35011`(앞2=시도) → 행정동 `35011760`(앞5=시군구)
- 예) 전주시 완산구=`35011`, 덕진구=`35012`. 쇠퇴진단 집계구(14자리)의 앞 8자리 = 행정동코드와 일치.
- 행안부 45체계와 조인하려면 **별도 매핑표 필요**(아직 안 만듦 → 아래 4번 후보).

## 3. 다시 뽑는 법 (갱신)
읍면동 개편 등으로 최신화가 필요하면 스크립트만 다시 돌리면 됨. **키는 코드에 없음(환경변수로 주입)**.

```powershell
# Windows PowerShell (이 폴더에서)
$env:SGIS_KEY="발급받은 consumer_key"
$env:SGIS_SECRET="발급받은 consumer_secret"
python sgis_admin_codes.py
```
```bash
# mac/linux
SGIS_KEY=... SGIS_SECRET=... python3 sgis_admin_codes.py
```
- 필요 라이브러리: `openpyxl` (이미 설치돼 있으면 추가 불필요)
- 전국 약 260회 호출이라 1~2분 소요.

## 4. 기술 메모 (막히면 참고)
- **SGIS API 서버 이전**: `sgisapi.kostat.go.kr` → **`sgisapi.mods.go.kr`** (국가데이터처 이관). 구 주소는 302 리다이렉트로 아직 작동하지만 스크립트는 새 주소 사용. `curl`로 테스트하면 `-L`(리다이렉트 따라가기) 필요.
- **인증**: `consumer_key`/`consumer_secret` 한 세트로 인구·사업체·주택·주소·경계 API 전부 공용. 쇠퇴진단이 쓰는 그 키 그대로 사용 가능.
- **핵심 엔드포인트**: `/OpenAPI3/addr/stage.json`
  - `accessToken`만 주면 → 시도 17개
  - `cd=35`(시도코드) → 하위 시군구
  - `cd=35011`(시군구코드) → 하위 행정동
  - 즉 3단계 재귀로 전국을 훑음.
- **세종시 처리**: 시군구 단계가 사실상 없어 `29010 세종시` 1개로 나옴 → 스크립트가 시군구=`29010 세종시`로 정상 처리.
- 엑셀에서 코드 열은 **텍스트 서식**(`@`)으로 저장 → 앞자리 0 보존, VLOOKUP/조인 안전. 헤더 고정 + 자동필터 걸어둠.
