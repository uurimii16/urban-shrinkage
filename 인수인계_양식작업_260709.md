## 1. 이번에 해결한 문제 2가지

### ① Z·T 점수가 "빈칸"으로 보이던 문제 (사용자 격노 지점)
- **원인**: openpyxl는 수식(`=(D3-AVERAGE...)/STDEV.P`)은 쓰지만 **계산된 값(캐시 `<v>`)을 파일에 안 넣는다.** 그래서 자동재계산을 안 하는 뷰어/수동계산 모드 엑셀에서 수식셀이 전부 빈칸으로 보였다. (수식 자체는 원본과 100% 동일했음 — 값이 안 박혀 있던 것뿐)
- **증거**: 원본 템플릿은 E3에 캐시값(`#DIV/0!` 등)이 저장돼 있는데, 생성물은 E3 캐시값=None 이었다.
- **해결**: `template_export.py`에
  - `_composite_cached_values(codes, vals, indicators)` — Z/T/부문/종합/등급을 파이썬(numpy)에서 엑셀 수식과 **동일 규칙**으로 계산.
  - `_inject_cached(xml, valmap)` — 저장된 xlsx의 워크시트 XML에서 수식셀 `<f>`에 계산값 `<v>`를 주입. **문자열 결과(O/X 등)는 셀에 `t="str"` 속성을 붙여야** 엑셀이 읽는다(안 붙이면 'O'를 숫자로 파싱하려다 파일 깨짐).
  - `save_wb(wb, target)` — `wb._tmpl_cache`(시트명→캐시맵)가 있으면 저장 후 주입. **모든 정본 저장은 `wb.save()` 대신 `TE.save_wb(wb, target)` 경유.**
- **결과**: 수식은 그대로 살아있고(지표값 바꾸면 자동 재계산), **열자마자 값이 꽉 참.**

### ② 시트 2개(계산방법+복합종합) → 원본 9시트 전부
`build_full_workbook`으로 9시트 조립.

### 원시 롱 7열 (1.1·1.2·2.1·2.2·2.3 공통)
`연도 | 집계구 | 연도집계구(=연도&집계구 문자열) | CODE | 값 | 항목명 | 행정동코드`
- 값: 숫자, 결측(N/A)은 `'N/A'` 텍스트.
- **항목명**(F열)은 VLOOKUP이 참조 안 하는 **설명열** → `_item_name(bucket, code)`로 패턴 생성:
  - to_in→'총인구', to_fa→'총사업체수', cp_bem→`종사자수(산업 0NN)`
  - ho_yr→건축밴드(1~5=1979이전/80년대/90년대/2000~04/2005~09, 6~=2010+(n-6)년)
  - ho_ar→연면적밴드(`_HO_AR_BANDS`)
  - in_age→`{lo}세이상~{lo+4}세이하`(총 1~22, 남 31~52, 여 61~82)

### 1.3 피벗 (건축년도별주택 오른쪽 I~AG)
- I2='행 레이블', J2~AC2=밴드20개(1979이전~2024년), AD2='총합계', AE2='집계구', AF2='노후건축물', AG2='전체건축물'.
- 행: 집계구별 밴드합. **AF(노후)=`=SUM(J:M)`**(1979이전~2004, 코드 1~4), **AG(전체)=`=SUM(J:AC)`**.
- 법적종합 AH/AI가 이 `AE:AG`를 VLOOKUP.

### 1 법적쇠퇴진단 종합 (핵심 수식 · 원본 그대로)
- A연번 B행정동(집계구[:8]) C집계구(14) D행정동명.
- **인구 연도열(E~)**: raw to_in의 **최근 30년 표본 전체**. 각 셀 `=VLOOKUP(E$3&$C4,'1.1. 총인구'!$C$2:$E${1+행수},3,FALSE)`.
- **사업체 연도열(U~)**: raw to_fa의 **최근 10년만**(`biz_years[-10:]` = 2014~2023). `=VLOOKUP(U$3&$C4,'1.2. 총사업체'!$C$2:$E${1+행수},3,FALSE)`.
- 증감률 R=`=IFERROR((Q4-MAX(E4:Q4))/MAX(E4:Q4)*100, 0)`, 해당 S=`=IF(R4<=-20,"O","X")`.
- 3년연속감소 T=`=IF(OR(AND(N4<M4,O4<N4,P4<O4),AND(O4<N4,P4<O4,Q4<P4)),"O","X")` (마지막 5개열 M~Q).
- 사업체 증감률 AE=`=IFERROR((AD4-MAX(U4:AD4))/MAX(U4:AD4)*100, 0)`, AF=`=IF(AE4<-5,"O","X")`, AG=3년연속(마지막5열).
- 노후 AH=`=VLOOKUP(C4,'1.3. 건축년도별주택'!$AE$3:$AG${2+피벗행수},2,FALSE)`, 전체 AI=col3, 비율 AJ=`=AH4/AI4*100`, AK=`=IF(AJ4>=50,"O","X")`.
- 부합개수 AL=`=IF(OR(S4="o", T4="o"), 1, 0) + IF(OR(AF4="o", AG4="o"), 1, 0) + IF(AK4="o", 1, 0)` (엑셀 문자비교는 대소문자 무시 → "O"="o").
- 쇠퇴지역 AM=`=IF(AL4>=2,"O","X")`.
- **행 수 = 복합종합과 동일(같은 집계구 집합, 정렬), 데이터 R4부터**(R1~3=헤더).

- 이전 인수인계 참조. `write_composite` + `_composite_cached_values`.

## 3. 코드 구조 (template_export.py)

핵심 진입점:

## 6. 남은 것 / 주의
- **Natural Break(AW열)**: 아직 비움(None). 등분위(AU)·등간격(AV)은 함수+캐시 완료. Jenks 경계 계산은 미구현(원본도 AW 채움 방식 불명확했음) — 필요 시 파이썬 Jenks로 경계 산출 후 VLOOKUP.
- **항목명 라벨**: 패턴 생성이라 극단 코드(0~4세, 100세+, 남/여 접두, 일부 연면적밴드) 문구가 원본과 미세하게 다를 수 있음. **VLOOKUP 미참조 설명열이라 계산엔 영향 없음.**
- **미푸시였다가 이번에 푸시**: `template_export.py`·`batch_build.py`·`app_v2.py`. 다른 세션이 편집중인 `batch_run.py`·`dong_names.py`·`행정구역코드/README.md`는 **커밋 금지**.
- 검증은 전주(로컬)로 함. **서울 25구 실데이터 최종확인은 웹에서** 스트리밍 돌려봐야 함.

## 8. 후속 마무리 (2026-07-14) — 남은 2·3 완료 + 잠복버그 수정

### ① Natural Break(AW열) Jenks 구현 (7번-2 완료)
- `_jenks_grades(at_values, k=10)` 신설: **순수 numpy DP**로 종합(AT)의 Natural Breaks(Jenks) 최적 경계를 구해 등급 매김. 엑셀엔 Jenks 함수가 없어 파이썬 계산값을 **정적 등급값**으로 AW(총합+3=49열)에 기록(수식이 아니라 값 → save_wb 캐시주입 대상 아님).
- **방향**: 종합 클수록 1등급(쇠퇴 심함), 최저 10등급. `keff=min(10, 고유값수)`로 스케일해 항상 1~10.
- 성능: n=1366 → **0.18초**, n=3366 → 0.61초 (빌드 20~70초 대비 무시 가능).
- AT 계산부를 `_composite_at()`로 추출해 `_composite_cached_values`(Z/T/부문/AT/등분위/등간격 캐시)와 `_composite_grades`(Jenks)가 공유 — 수식/캐시 산식과 100% 동일 규칙. `write_composite(grades=)`로 AW 기록, build_full_workbook·build_composite_workbook 둘 다 연결.

### ② 항목명 라벨 정밀 대조·수정 (7번-3 완료)
- 정본 `복합쇠퇴진단.xlsx` 원시 6시트의 CODE→항목명을 `_item_name`과 전수 대조.
- **불일치는 딱 2개(모두 in_age 성연령)**: `in_age_001` 원본 `4세이하`(↔생성 `0세이상~4세이하`), `in_age_021` 원본 `100세이상`(↔생성 `100세이상~104세이하`). 최저·최고 밴드는 하/상한이 없는 표기.
- 수정: `_item_name`의 in_age 분기에 `k==21 → '100세이상'`, `k==1 → '4세이하'` 추가(`k==22 연령미상` 유지). **ho_yr(20)·ho_ar(9)·cp_bem(17)·to_in·to_fa는 원래 완전 일치**.
- 참고: 항목명은 VLOOKUP 미참조 설명열 → 계산엔 영향 없음(형태만 원본과 정확히 일치시킴).

### ③ batch_build.py 잠복버그(NameError) 수정 — 중요

- 수정: `batch_build.py` 상단에 `import template_export as TE` 추가(순환참조 없음).
